# Data Types

## List

```python
from typing import List

mylist = ["apple", "banana", "cherry"]
```

* Ordered
* Changeable
* Allow Duplicates

## Tuple

```python
mytuple = ("apple", "banana", "cherry")
```

* Ordered
* Unchangeable
* Allow Duplicates

## Set

```python
myset = {"apple", "banana", "cherry"}
```

* Unordered
* Unchangeable
* Duplicates not allowed

## Dictionary

```python
thisdict = {
  "brand": "Ford",
  "model": "Mustang",
  "year": 1964
}
```

* Ordered (unordered until Python 3.7)
* Changeable
* Duplicates not allowed
